Эмуляторы

Anex86
Является одним из наиболее многофункциональных эмуляторов PC-98. Многие отмечают хорошую совместимость с играми. Главной его особеностью является поддержка реального или виртуального CD-дисковода.
T98-Next
Эмулятор японских компьютеров серии PC-98 для ОС Windows. Главной проблемой эмулятора является его не лучшая эмуляция звукового чипа PC-98, в результате чего музыка иногда может играть то быстрее, то медленнее.
Neko Project II
Это единственный эмулятор PC-98, который разрабатывается по сегодняшний день. Также это единственный эмулятор платформы, имеющий открытый исходный код.
